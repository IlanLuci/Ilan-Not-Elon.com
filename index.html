Asset Tracker

This is a portfolio tracking tool I built using Claude and Copilot. It brings multiple investment, cash, crypto, and debt accounts together in one place, so I can see both long-term trends and real-time performance in one place. It supports brokerage, Roth IRA, Traditional IRA, cash (in any ISO currency), crypto, debt (credit cards and other balances that net against your total), and watchlist account types, with daily close prices and live quotes pulled from Yahoo Finance. Accounts can be linked through Plaid for automatic transaction and balance syncing, and a spending-analysis view breaks down where the money actually goes.

Under the hood it uses C++ for the data aggregation, storage, and retrieval system — a multithreaded HTTP server with on-disk binary persistence, in-memory quote caching, and a parsed-data cache that skips re-reading files that haven't changed. Responses are gzip-compressed and support conditional (ETag) requests, while the static frontend throttles its polling when the market is closed or the tab is backgrounded. It's the first hobby project I've created using AI software development agents.

Hexholm

Hexholm is a real-time, multiplayer implementation of Catan for two to four players." Players spin up a table and invite friends with a room code or shareable link, or fill empty seats with AI opponents. It covers the full base game — building settlements, cities, and roads; development cards; the robber; longest road and largest army; and bank, harbor, and player-to-player trading — all enforced on the server so no one can cheat.

Under the hood, each game room is a Cloudflare Durable Object that holds the authoritative game state and coordinates every player over their own WebSocket, backed by a pure, deterministic rules engine." The frontend is a React app that reproduces the game's board and interface; the backend runs entirely on Cloudflare Workers with per-room state, automatic reconnection, rate limiting, an idle-takeover system that hands an abandoned seat to a bot, and an admin dashboard tracking players and match statistics.

Hexholm is another hobby project I built end-to-end with AI software development agents." The rules engine is fully test-covered, and empty seats are played by heuristic bots that trade with the bank and contest the special cards — a decent opponent for casual and learning players.
